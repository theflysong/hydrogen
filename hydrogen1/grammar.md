# Grammar

+号代表前面的内容可以出现1次或多次
*号代表前面的内容可以出现0次或多次
|代表或者
\[a-z\]代表a-z间的所有字母
\[1-9\]之类同理

ABC -> xxx

ABC代表单词token的类型
xxx代表这个单词的**产生式**

**产生式**是一种匹配规则
如果某个字符串符合ABC的**产生式**则这串字符串的单词类型为ABC

```
DecNumber -> ([1-9] | [0-9])* | 0
HexNumber -> 0(x|X)([0-9] | [a-f] |[A-F])+
OctNumber -> 0[0-7]+
BinNumber -> 0(b|B)[0-1]+
```
